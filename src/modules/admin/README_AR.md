# وحدة الإدارة (Admin Module)

هذا الملف يشرح الغرض من كل صفحة في قسم الإدارة.

## الصفحات الرئيسية:

- **Dashboard (لوحة التحكم):** تعرض نظرة عامة شاملة على أداء النظام، الإحصائيات الحالية، والنشاطات الأخيرة.
- **Users (المستخدمون):** تستخدم لإدارة جميع حسابات المستخدمين في النظام، بما في ذلك إضافة مستخدمين جدد، تعديل صلاحياتهم، أو تعطيل حساباتهم.
- **Modules (النماذج/الوحدات):** قائمة بالوحدات البرمجية أو الأقسام المتاحة في النظام وكيفية تفعيلها أو تخصيصها.
- **Plan Features (ميزات الخطط):** مخصصة لتحديد وإدارة الميزات المتاحة لكل خطة اشتراك (مثل الخطة المجانية، الاحترافية، إلخ).
- **Subscribers (المشتركون):** تعرض قائمة بكافة المستخدمين المشتركين في الخطط المدفوعة، مع تفاصيل تواريخ انتهاء الاشتراك وحالة الدفع.
