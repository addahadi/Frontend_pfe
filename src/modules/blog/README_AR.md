# وحدة المدونة (Blog Module)

هذا الملف يشرح الغرض من كل صفحة في قسم المدونة.

## الصفحات الرئيسية:

- **Articles (المقالات):** تعرض جميع المقالات المنشورة في النظام للعامة.
- **Article Editor (محيط المقالات):** واجهة مخصصة لكتابة مقالات جديدة أو تعديل المقالات القائمة، مع أدوات التنسيق وحفظ التغييرات.
- **Tags (الوسوم/التصنيفات):** تسمح بإدارة تصنيفات المقالات لتسهيل البحث عنها وتنظيمها.
